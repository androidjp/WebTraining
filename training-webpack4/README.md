## 安装
---
* 建议node 8.2以上版本
* 全局安装webpack
* `npm i webpack webpack-cli -D`
* 本地dev，需安装 webpack-dev-server: `npm i webpack-dev-server -D`

> 注意：用`yarn add webpack -D`实际上是本地安装，不是全局安装，并且`-D`是`--dev`的简写，所以是在`devDependencies{}`中装。全局安装命令是：`yarn global add <packageName>`
